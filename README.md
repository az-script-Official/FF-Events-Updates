## Installation | التثبيت

### Option 1: Direct Use | الطريقة الأولى: استخدام مباشر

1. Download the `index.html` file
2. حمّل ملف `index.html`
3. Open it in any modern browser (Chrome, Firefox, Safari, Edge)
4. افتحه في أي متصفح حديث (كروم، فايرفوكس، سفاري، إيدج)

### Option 2: GitHub Pages | الطريقة الثانية: صفحات جيت هب

1. Create a new GitHub repository
2. أنشئ مستودع جديد على GitHub
3. Upload the `index.html` file to the repository root
4. ارفع ملف `index.html` إلى جذر المستودع
5. Go to Settings > Pages > Source: Deploy from branch > Main > / (root)
6. اذهب إلى Settings > Pages > Source: Deploy from branch > Main > / (root)
7. Your site will be available at `https://yourusername.github.io/repo-name`
8. سيكون موقعك متاحاً على `https://yourusername.github.io/repo-name`

### Option 3: Local Server | الطريقة الثالثة: سيرفر محلي

```bash
# Using Python | باستخدام بايثون
python -m http.server 8000

# Using Node.js | باستخدام نود
npx serve .
```

## How to Use | طريقة الاستخدام

1. **Select a Region** - Click on any server button (ME, PK, BR, etc.)
   **اختر سرفر** - اضغط على أي زر سرفر (ME, PK, BR, إلخ)

2. **View Events** - Browse splash banners and news cards
   **عرض الأحداث** - تصفح بنرات البداية وبطاقات الأخبار

3. **Expand Details** - Click on any card header to see event details (start/end time, links)
   **عرض التفاصيل** - اضغط على عنوان أي بطاقة لرؤية تفاصيل الحدث (وقت البداية/النهاية، الروابط)

4. **Download Banners** - Click the download button to save event images
   **تحميل البنرات** - اضغط على زر التحميل لحفظ صور الأحداث

5. **Switch Language** - Use the language switch button to change event display language
   **تبديل اللغة** - استخدم زر تبديل اللغة لتغيير لغة عرض الأحداث

6. **Go Back** - Click the back button to return to the server selection
   **رجوع** - اضغط على زر الرجوع للعودة إلى اختيار السرفر
